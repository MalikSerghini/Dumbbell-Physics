## About The Project

A small project based on the physics engine react-three/rapier.

Physics Engine: <a href="https://github.com/pmndrs/react-three-rapier">React-Three-Rapier<a/>

## Setup

Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev
```
